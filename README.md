# DNSpropag

```
DNSpropag is a command line tool for verifying DNS propagation.
```

## Requirements
```
pip3 install -r requirements.txt
```

## Usage

```
Usage: dnspropag.py [OPTIONS]

Options:
  --domain
  --dnstype
  --help
```

## Example

```
python3 dnspropag.py --domain python.org --dnstype A
```
